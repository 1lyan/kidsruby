Что такое глоссарий (словарь)?

Глоссарий представляет собой набор определений для слов, которые относятся к конкретному субьетку. Это похоже на тематический словарь.

Глоссарий

character: Одна буква, число или символ.

class: Класс - это план для создания обьектов. Объект данного класса называется "экземпляром" этого класса. Класс может содержать атрибуты, методы и даже другие классы!

method: Методы позволяют классам делать что-либо. Они являются набором инструкций и могут возвращать значениея.

parameter: Тип переменной в круглых скобках, который накладывает ограничение или ограничения на метод на который он действует.

string: Строка - это любое количество символов (буквы, цифры, знаки препинания) окружено кавычками. Как одиночные, так и двойные кавычки используются для создания строк (Источник: Why's Poignant Guide)

icon: Картинка на экране которая отображает определенный файл, каталог, окно, опцию или программу. (Источник: The Free Dictionary)

pixel: любое число из множества очень маленьких элементов изображения, которые составляют его на визуальном дисплее (Источник: The Free Dictionary)

attribute: Свойство или характеристика Обьекта. Например, свойства человека будут имя, возраст, вес и т.п... (Источник: Computing Students)

variable: Символ используется для обозначения данных, назначенных переменной. Например, в x = 50, "x" - это переменная, а "50" - это данные которые содержатся в ней.

object: Элемент, который можно выбрать и маневрировать, например как графику на экране. Эти объекты могут включать данные и процедуры, необходимые для работы с этими данными.(Источник: The Free Dictionary)

.png: Позиции для "Портативной Сетевой Графики" являющиеся файловым форматом сжатия изображений, всего лишь воображаемый термин для того что бы сделать файл изображения менее обьемным без потери качества.

instantiate: Создание переменной для класса. Например, создание экземпляра переменной класса "Cat" может быть "My Cat".
