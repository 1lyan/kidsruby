Команды Сферо

Назад к Роботам Назад к Сферо

Следующий список команд может использоваться с вашим Сферо:

Sphero.start(port) do...end

Подключается к Сферо и начинает выполнять команды между 'do' и 'end'

port - как подключится к определенному Сферо

скорость вращения, заголовок

Заставь Сферо начать катиться в определенном направлении

скорость - число между 0 и 255

заголовок - число от 0 до 359 для какого либо направления

стоп

Заставь Сферо остановиться

heading = h

Заставь Сферо сменить направление движения когда он в процессе

h - число от 0 до 359 для какого либо направления

rgb r, g, b

Изменить у Сферо LED цвета

r - число между 0 и 255

g - число между 0 и 255

b - число между 0 и 255

keep_going duration

Сообщает, что Сферо продолжает делать то, что уже делает в течение определенного времени

duration - количество секунд для продолжения

пинг

Пинги Сферо, чтобы узнать, отвечает ли он

ВПЕРЕД

Постоянное значение, равное 0

НАЗАД

Постоянное значение, равное 180

ПРАВО

Постоянное значение, равное 90

ЛЕВО

Постоянное значение, равное 270

Существует еще больше команд, но эти самые важные. Для детальной информации, загляните в документацию гема Sphero на Github.
